Timestamp MS

FreeCodeCamp - API Project

A microservice that take timestamp/natural date and return natural date and timestamp that equal the given input.

Try the live version.

Example usage:

  • https://aalakkad-api-timestamp.herokuapp.com/December%2015,%202015
  • https://aalakkad-api-timestamp.herokuapp.com/1450137600

Example output: 

{
  "unix": 1450137600,
  "natural": "December 15, 2015"
}
